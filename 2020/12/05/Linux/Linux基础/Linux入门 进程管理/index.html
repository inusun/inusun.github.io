进程管理

什么是进程

  1. 在Linux中，每个程序都有一个进程，每一个进程都有一个id号
  2. 每一个进程都会有一个父进程
  3. 进程可以有两种存在方式（前台/后台）
  4. 一般的话，服务都是在后台运行，基本的程序都是在前台运行

常用命令

ps 查看正在运行的进程

选项：

  • -a：向是所有正在运行的进程
  • -u：以用户分组
  • -x：显示后台运行进程的参数
